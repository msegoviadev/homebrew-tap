# msegoviadev/homebrew-tap

Homebrew tap for [spec-mind](https://github.com/msegoviadev/spec-mind).

## Installation

```bash
brew tap msegoviadev/tap
brew install spec-mind
```

Or directly:

```bash
brew install msegoviadev/tap/spec-mind
```

## Usage

```bash
spec-mind convert openapi.yaml
spec-mind sync ./api-specs/
spec-mind validate openapi.yaml
```

For more information, see [spec-mind](https://github.com/msegoviadev/spec-mind).
